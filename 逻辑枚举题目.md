## Q1

警察局抓了a,b,c,d四名偷窃嫌疑犯，其中有一人是小偷。审问中：
a说：“我不是小偷”。
b说：“c是小偷”。
c说：“小偷肯定是d”。
d说：“c冤枉人”。
现在已经知道四人中三人说的是真话，一人说的是假话。问到底谁是小偷？

#### 分析
根据题目所述，四人的话可转换为以下逻辑表达式：

- a: thief != a
- b: thief == c
- c: thief == d
- d: thief != d

枚举thief可取的情况，满足三条逻辑表达式为真即可。

```
for thief in ['a','b','c','d']:
    if(thief != 'a') + (thief == 'c') + (thief == 'd') + (thief !='d') == 3:
        print("小偷是：%s " % thief) ##c
```
